## About this project:

Backend proyect about a database of a video store, when you can create users, modify its properties and get rents of films.

## Requirements:

* Register user.
* Obtain user data.
* User login.
* Modify user data.
* List all the films.
* Search films by title.
* Generate the rent of a film.
* List all the rents.
